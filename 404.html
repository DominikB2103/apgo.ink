Questions

Built for founders who want momentum, not excuses.

Do you only make websites?

No. The website is the visible layer. Wylo also helps organize the offer, business tools, customer flow, content direction, and growth plan.

Can this run on GitHub Pages?

Yes. This site is built as static HTML, CSS, and JavaScript. No database is needed. Contact uses a mailto-based flow unless you connect an external form service later.

Can we add generated imagery later?

Yes. The current build uses custom front-end visuals and has space for art-directed image assets. The included asset prompt sheet gives directions for future generated images.

Is the style minimal or maximal?

It can support either. Wylo's standard is not minimalism or maximalism. The standard is taste: clear hierarchy, strong composition, useful motion, and no childish gimmicks.
