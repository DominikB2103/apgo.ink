# Tannenhof Mayer Art — apgo.ink/art

This folder is a fully static art page for GitHub Pages.

## Deploy

Copy the entire `art/` folder into the root of your GitHub Pages repository. It will be available at:

`https://apgo.ink/art/`

No server, build step, database, or package installation is required.

## Edit

- Main page: `art/index.html`
- Styles: `art/assets/css/styles.css`
- Interactions: `art/assets/js/main.js`
- Social preview and icons: `art/assets/img/`

## Built-in features

- Monochrome editorial / neo-brutalist visual system
- Organic shapes and small color signal system
- Accent color cycling
- Generative gesture canvas
- Interactive all-art permission chips
- Shuffleable living wall
- Prompt generator
- Art weather board
- Print-as-catalog-sheet mode
- Copy email / copy prompt buttons
- Fully responsive static layout
