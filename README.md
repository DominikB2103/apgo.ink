# Tannenhof Mayer Website

This is a static, GitHub Pages-ready website for **Tannenhof Mayer**.

## Update included
This revision overhauls the visual system to be more serious and readable:
- Replaced the more playful display font with **Source Serif 4** for headings.
- Tightened typography and reduced oversized hero copy.
- Rebuilt the charts so they use coherent sample data and readable axes.
- Replaced the abstract community graphic with a structured operating map.
- Updated all Discord links to the live server:
  - `https://discord.gg/HPDEQh9Pgc`

## Deploy
Upload all files to your GitHub Pages repository root.

## Main files
- `index.html` — landing page
- `pages/thesis.html` — founder thesis
- `pages/community.html` — community standards
- `pages/legal.html` — disclosure page
- `assets/css/*` — modular styles
- `assets/js/*` — modular JavaScript and chart rendering

## Notes
- The charts use **illustrative sample data** for front-end presentation.
- No server is required.
