# Wylo website

Static GitHub Pages site for Wylo.

## Deploy

Upload the files in this folder to a GitHub repository and enable GitHub Pages from the repository settings.

The included `CNAME` points the site to:

```txt
apo.ink
```

## Files

- `index.html` — page structure and copy
- `styles.css` — layout, responsive styling, and motion
- `script.js` — hero state animation, package selector, reveal behavior
- `assets/favicon.svg`
- `assets/og-image.svg`
- `CNAME`
