CONTINUUM REVIEW — STATIC GITHUB PAGES SITE

Upload every file and folder in this ZIP to your GitHub Pages repository root.
Open index.html locally or through GitHub Pages.

To add a new document:
1. Copy documents/template/ into a new folder inside documents/.
2. Rename the folder, for example documents/my-new-note/.
3. Edit that folder's index.html.
4. Add the new entry to the documents array near the top of script.js so it appears in search and filters.

No npm install, no build step, and no external dependency is required.
