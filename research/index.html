Document subfolder

Upload each document as its own folder.

Place new HTML documents under documents/your-document-name/index.html. Start from the included template, replace the metadata and article body, then add a matching entry inside assets/js/site.js.

Open the template

  1. Duplicate documents/_template.
  2. Rename the folder using lowercase hyphenated words.
  3. Edit the title, abstract, tags, figures, and article body.
  4. Add the new document to the archive array in site.js.
