Analysis / Fixed points

Banach contraction principle

In a complete metric space, every contraction has exactly one fixed point, and iterating the contraction from any starting point converges to it.

d(Tx, Ty) ≤ q d(x,y),   0 ≤ q < 1

Linear algebra / Operators

Spectral theorem for real symmetric matrices

Every real symmetric matrix admits an orthonormal basis of eigenvectors, allowing the matrix to be diagonalised by an orthogonal change of coordinates.

A = QΛQT,   QTQ = I
