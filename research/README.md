# Asterion Review static site

Upload every file in this folder to GitHub Pages. The site is static and does not require a package manager, build step, or server-side code.

## Edit the archive

1. Copy `documents/template/` and rename the copied folder.
2. Edit the new folder's `index.html`.
3. Add a matching entry in `assets/js/data.js` so it appears in search.

## Main files

- `index.html` — landing page, sections, theorem carousel, document search.
- `assets/css/styles.css` — full visual system and article styling.
- `assets/js/data.js` — searchable archive entries.
- `assets/js/main.js` — navigation, carousel, search, and filters.
- `documents/template/index.html` — complete article template.
