# APTO.INK package

The root of this folder is already a static website. Uploading it as-is will serve `index.html` first.

Optional Next.js source is included in `app/`, `components/`, `sections/`, `data/`, `lib/`, and `styles/` for future development.

For direct static upload, the important public files are:

- `index.html`
- `demos/index.html`
- `demos/bakery/index.html`
- `demos/garage/index.html`
- `demos/municipality/index.html`
- `demos/clinic/index.html`
- `assets/`
- `legal/`
- `404.html`
